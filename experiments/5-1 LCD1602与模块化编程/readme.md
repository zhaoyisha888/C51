### 模块化注意
1. .c文件:函数、变量的定义
2. .h文件:可被外部调用的函数、变量的声明

   - 任何自定义的变量、函数在调用前必须有定义或声明(同一个.c)
   - 使用到的自定义函数的.c文件必须添加到工程参与编译
   - 使用到的.h文件必须要放在编译器可寻找到的地方(工程文件夹根目录、安装目录、自定义)

3. **C语言预编译**
   ![alt text](image.png)

   **对程序某些部分是否预编译进行选择**
   ```c
    #ifndef __AT89X52_H__  //判断是否定义过
    #define __AT89X52_H__  //如果没有定义过，就定义
    //对应C文件的函数声明
    ……
    #endif                 //定义结束
   ```
>这里将之前数码管模块化到common文件夹中
